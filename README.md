# FarmChecklist
Telegram-бот для управления фермой кейсов в CS2, с базой SQLite и командами для управления

# Функции
- Просмотр информации об аккаунтах: ник, ссылка на профиль, код друга и т.д.
- Взаимодействие с базой данных (добавление, изменение, удаление аккаунтов)
- Защита опасных команд паролем (устанавливается в config.py)
